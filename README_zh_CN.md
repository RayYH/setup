# Setup

[English](README.md) | 简体中文

---

该仓库用于快速设置 **我的** MacOS 开发环境。

## 安装

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/rayyh/setup/master/install.sh)"
```

## 更新

```bash
$ upgrade_set_up
```

## 同步全局配置文件

```bash
# 同步全局配置文件
$ sync_set_up_configs

# 清理备份配置文件
$ clean_set_up_backup_files
```
